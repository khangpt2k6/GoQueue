GoQueue

A high-throughput message broker in Go. Built from scratch for local scale experiments: TCP and gRPC transport, partitioned topics, WAL durability, and observability with Prometheus + Grafana + OpenTelemetry.

GitHub Releases Grafana

Core Capabilities

  • Custom binary TCP protocol and typed gRPC API
  • Partitioned topics with key-based routing
  • Write-ahead log for replay and recovery
  • Consumer groups with partition-aware offsets
  • Batch publish/fetch for higher throughput

Local Stack

Docker Compose starts 3 independent broker nodes with shared observability.

docker compose up --build

Broker

Run a broker manually for fast local development.

go run ./cmd/broker --tcp-addr=:9090 --grpc-addr=:9095 --metrics-addr=:2112
